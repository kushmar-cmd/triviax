בוט טלגרם לטריוויה בעברית עם 8 קטגוריות ושני מצבי משחק, מבוסס Claude AI.

---

## קטגוריות

| קטגוריה | |

## מצבי משחק

- **🎮 סולו** — `/play` — שחקן אחד בוחר קטגוריה + קושי, עונה לבד
- **👥 קבוצתי** — `/trivia` — כולם מתחרים, מי שלוחץ ראשון מנצח (טיימר 20 שניות)

---

## התקנה

```bash
pip install -r requirements.txt
```

## משתני סביבה

```
BOT_TOKEN=123456:ABC-DEF...
ANTHROPIC_API_KEY=sk-ant-...
```

## הפעלה

```bash
python bot.py
```

---

## יצירת הבוט ב-BotFather

1. `@BotFather` → `/newbot`
2. קבל TOKEN
3. הגדר פקודות עם `/setcommands`:

```
play - 🎮 משחק סולו
trivia - 👥 שאלה קבוצתית
score - 📊 הניקוד שלי
top - 🏆 טבלת מובילים
help - ❓ עזרה
```

4. **חשוב לקבוצות:** הפעל `/setprivacy` → `Disable` כדי שהבוט יקבל עדכונים בקבוצה

---

## פריסה מהירה — Replit

1. פתח replit.com → צור Repl חדש → Python
2. העלה `bot.py` + `requirements.txt`
3. Secrets: הוסף `BOT_TOKEN` ו-`ANTHROPIC_API_KEY`
4. לחץ Run

---

## איך עובד מצב קבוצתי

1. מישהו מקליד `/trivia` בקבוצה
2. בוחרים קטגוריה + קושי
3. השאלה מופיעה עם **טיימר 20 שניות**
4. **כל חברי הקבוצה** רואים את הכפתורים
5. מי שלוחץ ראשון — מנצח (ומקבל נקודה)
6. אם לא ענו — הבוט חושף את התשובה אוטומטית
